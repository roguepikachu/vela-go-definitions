defkit API Reference

The Go fluent API for building KubeVela X-Definitions programmatically. Replace CUE/YAML templates with type-safe Go code that compiles to standard KubeVela definitions.

Definition Builders

Top-level constructors that create ComponentDefinition, TraitDefinition, PolicyDefinition, and WorkflowStepDefinition objects. Each builder is registered globally via defkit.Register() in an init() function.

Go
func MyComponent() *defkit.ComponentDefinition {
    return defkit.NewComponent("my-component").
        Description("A custom workload component").
        Workload("apps/v1", "Deployment").
        Params(imageParam, replicasParam).
        Template(buildTemplate)
}
CUE
// Generated CUE output
#my-component: {
    type: "component"
    name: "my-component"
    description: "A custom workload component"
    workload: {
        apiVersion: "apps/v1"
        kind: "Deployment"
    }
}

Parameter Builders

Fluent builders for defining typed input parameters. Each builder supports chaining for required flags, defaults, descriptions, and validation constraints.

Go
image := defkit.String("image").
    Required().
    Description("Container image to deploy")

replicas := defkit.Int("replicas").
    Default(1).
    Description("Number of pod replicas")
CUE
parameter: {
    image: string
    replicas: *1 | int
}
// image is required (no default)
// replicas defaults to 1

Template Methods

Methods on *defkit.Template that control what the definition outputs — primary resources, patch containers, and auxiliary outputs.

Go
func buildTemplate(tpl *defkit.Template) {
    resource := defkit.NewResource("apps/v1", "Deployment")
    // ... configure resource
    tpl.Output(resource)
    tpl.Outputs("svc", serviceResource)
}
CUE
output: {
    apiVersion: "apps/v1"
    kind: "Deployment"
    // ... resource spec
}
outputs: svc: {
    apiVersion: "v1"
    kind: "Service"
}

Resource Builders

Builders for constructing Kubernetes resource objects. Resources are typed by apiVersion and kind, with fluent setters for metadata, spec fields, and nested structures.

Go
resource := defkit.NewResource("apps/v1", "Deployment").
    Set("metadata.name", vela.Name()).
    Set("spec.replicas", defkit.Reference("parameter.replicas")).
    Set("spec.template.spec.containers[0].image",
        defkit.Reference("parameter.image"))
CUE
output: {
    apiVersion: "apps/v1"
    kind:       "Deployment"
    metadata: name: context.name
    spec: {
        replicas: parameter.replicas
        template: spec: containers: [{
            image: parameter.image
        }]
    }
}

Value Expressions

Primitives for embedding literal values and parameter references into resource fields. These are the building blocks of dynamic resource construction.

Go
defkit.Lit("always")
defkit.Lit(3)
defkit.Reference("parameter.image")
defkit.Reference("context.name")
defkit.Concat(vela.Name(), defkit.Lit("-svc"))
CUE
"always"
3
parameter.image
context.name
context.name + "-svc"

VelaCtx

The defkit.VelaCtx() accessor provides runtime context values injected by KubeVela — application name, namespace, component name, and more.

Go
vela := defkit.VelaCtx()

vela.Name()
vela.Namespace()
vela.AppName()
vela.ClusterName()
CUE
context.name
context.namespace
context.appName
context.clusterName

Health & Status DSL

Builders for defining how KubeVela evaluates component and trait health. Conditions map resource field values to healthy/unhealthy states.

Go
defkit.Health().
    HealthyWhen(
        defkit.Field("status.readyReplicas").
            Equals(defkit.Reference("spec.replicas")),
    ).
    Message("Waiting for replicas to be ready")
CUE
isHealth: output.status.readyReplicas ==
          output.spec.replicas
message: "Waiting for replicas to be ready"

Full Examples

Complete, working component definitions showing the full defkit pattern from parameter declaration through template output and registration.

Go
func WebService() *defkit.ComponentDefinition {
    image := defkit.String("image").Required().
        Description("Container image")
    port := defkit.Int("port").Default(8080).
        Description("Service port")

    return defkit.NewComponent("webservice").
        Description("HTTP web service component").
        Workload("apps/v1", "Deployment").
        Params(image, port).
        Template(webServiceTemplate)
}

func webServiceTemplate(tpl *defkit.Template) {
    vela := defkit.VelaCtx()
    deploy := defkit.NewResource("apps/v1", "Deployment").
        Set("metadata.name", vela.Name()).
        Set("spec.replicas", defkit.Lit(1)).
        Set("spec.template.spec.containers[0].image",
            defkit.Reference("parameter.image"))
    tpl.Output(deploy)
}

func init() { defkit.Register(WebService()) }
CUE
// Equivalent CUE template
parameter: {
    image: string
    port:  *8080 | int
}

output: {
    apiVersion: "apps/v1"
    kind:       "Deployment"
    metadata: name: context.name
    spec: {
        replicas: 1
        template: spec: containers: [{
            image: parameter.image
        }]
    }
}
